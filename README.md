## вңЁ Features

- вҡЎ **One-click scan** вҖ” fetches all 125 targets via Torn API simultaneously
- рҹҺҜ **Smart auto-sort** вҖ” players ranked by attack priority automatically:
  1. `OKAY` вҖ” attackable right now, always at the top
  2. `HOSPITAL` вҖ” sorted by **shortest time remaining first**
  3. `JAIL` вҖ” sorted by **shortest time remaining first**
  4. `TRAVELING` вҖ” listed after
  5. Unknown / unloaded вҖ” at the bottom
- вҸұпёҸ **Live countdown timers** вҖ” see exactly how long until a player is out of hospital or jail (e.g. `HOSPITAL [4m 32s]`)
- вҡ”пёҸ **One-click attack button** вҖ” opens Torn attack page directly for each player
- рҹ”Қ **Search + filter** вҖ” filter by name/ID or status (Okay / Hospital / Jail / Traveling)
- рҹ“Ҡ **Stats bar** вҖ” live count of each status group as scan runs
- рҹ–ҘпёҸ **Zero setup** вҖ” single `.html` file, no server, no install, no dependencies
- рҹ”’ **Privacy first** вҖ” your API key stays in your browser, never sent anywhere else

## рҹ”‘ API Key Setup

1. Log in to **[Torn City](https://www.torn.com)**
2. Go to **Settings вҶ’ API Key**
3. Create a key with **`Public` access level** (minimum required)
4. Paste the key into the dashboard when scanning

> вҡ пёҸ **Security tip:** Use a `Public` access key only вҖ” this tool only needs `profile` data. Never share your full-access key.

## рҹ“Ў How It Works

```
Your Browser
    в”Ӯ
    в”ңв”Җв”Җ Loads index.html (125 player IDs pre-loaded)
    в”Ӯ
    в”ңв”Җв”Җ On "SCAN ALL" click:
    в”Ӯ       в””в”Җв”Җ Fetches each player via:
    в”Ӯ           GET https://api.torn.com/user/{ID}?selections=profile&key={YOUR_KEY}
    в”Ӯ
    в”ңв”Җв”Җ Extracts:
    в”Ӯ       в”ңв”Җв”Җ level
    в”Ӯ       в”ңв”Җв”Җ status.state        (Okay / Hospital / Jail / Traveling)
    в”Ӯ       в”ңв”Җв”Җ status.description  (e.g. "In hospital for 15 mins")
    в”Ӯ       в”ңв”Җв”Җ states.hospital_timestamp
    в”Ӯ       в”ңв”Җв”Җ states.jail_timestamp
    в”Ӯ       в””в”Җв”Җ last_action.relative
    в”Ӯ
    в””в”Җв”Җ Auto-sorts by priority + time remaining вҶ’ renders table
```

Requests are batched **5 at a time** with a 600ms delay between batches to respect Torn's API rate limits.
